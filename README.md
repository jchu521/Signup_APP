# Login App  (still working on it)
React Native + Node.js.


## Features

* create account
* Login account

## React Native
### Installation
`npm install`

### How to start

`$ react-native run-ios // run on ios` 

`$ react-native run-android //run on android`

## Server
### Installation && start
`cd Server`
`npm install`
`nodemon index.js`
